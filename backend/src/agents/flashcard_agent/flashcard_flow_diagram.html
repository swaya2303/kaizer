üîç Key Differences Between Testing & Learning Flows

Testing Flashcards
Focus: Multiple choice questions for assessment
Content: Text-based questions with 4 options
Processing: Text chunking for large documents
Output: Interactive MCQ cards with JavaScript
Agent: TestingFlashcardAgent
Template: MCQ-specific Anki templates
Learning Flashcards
Focus: Front/back cards for learning
Content: Concept-based with images
Processing: Chapter-based parallel processing
Output: Traditional flashcards with media
Agent: LearningFlashcardAgent
Template: Standard Anki card templates
